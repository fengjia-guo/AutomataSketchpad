# AutomataSketchpad
[https://automata-sketchpad.vercel.app](https://automata-sketchpad.vercel.app)

## Getting Started
### Clone the repository
```bash
git clone https://github.com/fengjia-guo/AutomataSketchpad.git
cd AutomataSketchpad
```
### Install dependencies
```bash
pnpm install
```
### Run
```bash
pnpm run dev
```
